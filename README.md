# Shreshth Mishra — Personal Portfolio

A modern, glassy, pastel-colored personal portfolio created for BCSE203E Web Programming.

## Included
- Responsive portfolio webpage
- Resume-based content sections
- Step-by-step documentation
- Learning reflections
- GitHub Pages deployment notes

## Files
- `index.html` — main webpage
- `styles.css` — design and layout
- `script.js` — interactivity
- `documentation.md` — assignment write-up

## Deployment
1. Create a GitHub repository.
2. Push these files to the repository.
3. Enable GitHub Pages from the repository settings.
4. Select the `main` branch and root folder.
5. Visit the generated GitHub Pages URL.

## Notes
The project is intentionally built without placeholder content. It uses the provided resume text and can be extended later with a photo, screenshots, or additional sections if needed.
