## 2. Project Summary
This project is a responsive personal portfolio webpage designed to present my profile, education, skills, projects, experience, certifications, and contact details in a polished and professional format.

The design uses a glassmorphism-inspired style with pastel colors, soft gradients, rounded cards, and clear typography to maintain readability while still feeling modern and visually appealing.

## 3. Step-by-Step Development Process
### Step 1: Planning the structure
I first decided the main sections required for the assignment:
- Home / introduction
- About
- Skills
- Experience
- Projects
- Education and certifications
- Contact information

### Step 2: Collecting content
I used the resume text provided for the portfolio content so that the webpage reflects real academic and professional information.

### Step 3: Designing the visual style
I chose a pastel palette with glass-like panels, blur effects, soft shadows, and floating gradient orbs to create a premium but calm appearance.

### Step 4: Building the webpage
I created the webpage using:
- HTML for structure
- CSS for styling and layout
- JavaScript for interactivity

### Step 5: Making it responsive
The layout was adjusted for mobile, tablet, and desktop screens so the portfolio remains readable on all devices.

### Step 6: Adding interaction
I added smooth scrolling, active navigation highlighting, a mobile menu, and scroll reveal animations.

### Step 7: Reviewing and polishing
I checked spacing, contrast, text sizes, and card balance to make the page clean and professional.

### Step 8: Preparing for GitHub Pages
The site is structured as a static webpage so it can be deployed easily using GitHub Pages.

## 8. Importance of a Personal Webpage
A personal webpage is important in the modern digital world because it creates a strong online presence and helps build professional identity. It acts as a centralized place to share academic background, technical skills, projects, and achievements.

For students and professionals, it also makes sharing work easier during internships, interviews, networking, and collaborations. A well-designed portfolio shows not only skills but also creativity, communication, and attention to detail. It can leave a stronger impression than a resume alone because it presents a more complete and interactive view of a person’s work.
